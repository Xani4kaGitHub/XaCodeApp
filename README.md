# XaCode Desktop

XaCode — локальный AI‑агент для разработки под Windows. Он работает с файлами проекта, терминалом, HTTP, базами данных и Docker через управляемый каталог инструментов и отдельные разрешения для каждого проекта.

## Что входит в версию 1.11.0

- динамическое включение и отключение инструментов: модель не получает схемы отключённых инструментов и не тратит на них контекст;
- строгая проверка аргументов инструментов и структурированные результаты;
- VerificationPipeline и ProtectionSystem перед завершением задачи;
- память с ограничением контекста и сохранением сессий;
- Windows Desktop UI с профилями моделей и проектными разрешениями;
- зашифрованное хранение API‑ключей через Electron `safeStorage`/Windows DPAPI;
- NSIS‑установщик с выбором каталога, ярлыками рабочего стола и меню «Пуск».

## Где хранятся данные

Все пользовательские данные находятся вне репозитория, в `%USERPROFILE%\.xacode` (обычно `C:\Users\<пользователь>\.xacode`). Переменная `XACODE_HOME` позволяет выбрать другой каталог.

| Путь | Содержимое |
| --- | --- |
| `settings.json` | настройки и зашифрованные API‑ключи |
| `conversations.json` | история чатов |
| `config.env` | настройки старого CLI/bot‑режима; не используется Desktop для API‑ключей |
| `projects\<id>` | память, todo, бэкапы, снимки и строгие sandbox‑каталоги проекта |
| `skills` | пользовательские skills |
| `logs` | журналы приложения |
| `workspaces` | проекты, созданные кнопкой «Новый проект» |

При первом запуске `settings.json` и `conversations.json` автоматически копируются из старого Electron userData‑каталога, если новые файлы ещё не существуют. API‑ключ в `settings.json` не хранится открытым текстом: расшифровать его может только тот же пользователь Windows.

## Разработка

Требуется Node.js 20+.

```powershell
npm.cmd install
npm.cmd run dev
```

Проверки:

```powershell
npm.cmd test
npm.cmd run test:ui
npm.cmd run test:verification
```

## Windows‑сборка

```powershell
npm.cmd run dist:win
```

Установщик создаётся в каталоге `release`. Этот каталог, зависимости, секреты, локальное состояние и логи исключены из Git.
